## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Ali-Maia/clubeDoLivro.git
```

Entre no diretório do projeto

```bash
  cd clubeDoLivro
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
